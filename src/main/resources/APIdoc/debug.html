1.1. 买家发起仲裁

URL: http://localhost:10080/[buyerAftermarket]/[aftermarketStart]

Type: POST

Author: zhouxv 2021/7/26.

Content-Type: application/json; charset=utf-8

Description: 买家点击售后按钮，进入售后页面，点击提交，调用此接口

Body-parameters:

false
